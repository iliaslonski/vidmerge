vidmerge · version "good enough 01"
Xiaomi camera clip merger · by @slonski

Склейка минутных видео с камер Xiaomi в один файл на Mac Инструкция по применению скрипта vidmerge.sh

С приветом собаке Агате, без которой склеивать было бы нечего.

Домашние камеры Xiaomi (C300 и аналогичные) пишут видео на SD-карту короткими файлами по одной минуте каждый. Если нужно получить одно цельное видео — например, час или два — эти кусочки надо склеить. Скрипт vidmerge.sh делает это автоматически: без потери качества и очень быстро (несколько секунд на час видео).

Какие камеры подходят Все домашние smart-камеры Xiaomi / Mi / Mijia, которые работают с приложением Mi Home и пишут видео на SD-карту минутными файлами. Среди типичных моделей: Xiaomi C300, C400, C500 Pro, Mi Home Security Camera 360°, Mi Outdoor CW400. Главный признак совместимости — если открыть SD-карту, на ней есть папка MIJIA_RECORD_VIDEO с подпапками вида 2026053014 (год+месяц+день+час): если такая папка есть, скрипт сработает.

Что понадобится

  • Mac
  • Камера Xiaomi с установленной SD-картой и приложение Mi Home на телефоне
  • Минут 15 на первичную настройку — её делать один раз навсегда
  • Несколько минут каждый последующий раз, когда нужно склеить новые видео
  • Способ подключить SD-карту к Маку (картридер или переходник — если хотите читать с карты напрямую)

Что такое скрипт. Это текстовый файл с инструкциями, которые компьютер выполняет одну за другой, — такие файлы запускают из Терминала. В отличие от обычных приложений (тех, что в папке Applications с иконкой), скрипт не требует установки: хранится там, где его положили, и запускается оттуда же.

Безопасно ли запускать. vidmerge.sh делает ровно одну вещь — берёт .mp4-файлы в указанной папке и склеивает их через ffmpeg. Не лезет в интернет, не трогает другие папки, не отправляет никаких данных. Сам файл — обычный текст: можно открыть его в любом текстовом редакторе (например, в TextEdit, который есть в любом Mac), и весь код будет перед вами.

Папка для видео

Скрипт не привязан к конкретной папке. Вы сами выбираете, куда положить минутные файлы — это может быть:

  • Новая папка на Рабочем столе («Видео с камеры», «Запись 30 мая» — любое название)
  • Папка внутри Документов
  • Любое другое место, где вам удобно

Главное правило: минутные файлы и сам скрипт vidmerge.sh должны лежать в одной папке. И в этой папке не должно быть никаких других видео-файлов от другой камеры — только текущая серия. После склейки итоговое видео тоже появится в этой же папке.

Совет Называйте папку по смыслу события, которое записано — например, Лидл, Уход_на_прогулку, 2026-05-30_визит_сантехника. Скрипт автоматически возьмёт имя папки как метку для итогового файла: оно попадёт и в имя файла (merged_..._Лидл.mp4), и в метаданные видео (видны в QuickTime/VLC и в поиске Spotlight). Так через год по списку файлов будет понятно, что именно записано в каждом видео.

Этап 1. Установка программ

Эти шаги нужно сделать только один раз — после этого можно сразу переходить к склейке.

1.1. Откройте Терминал

Терминал — это окошко, через которое можно давать компьютеру команды текстом. Найти его так:

  1. Откройте поиск Spotlight:
    • стандартный способ — нажмите Cmd + Пробел. Это шорткат по умолчанию;
    • если шорткат не сработал — кликните по иконке лупы в правом верхнем углу экрана, в строке меню рядом с часами;
    • запасной способ, если иконки лупы тоже нет: Finder → Программы (Applications) → Утилиты (Utilities) → Терминал (Terminal), и пропустите шаги 2–3.
  2. В появившейся строке Spotlight наберите Terminal и нажмите Enter.
  3. Откроется окно с курсором — это и есть Терминал.

Как работать с Терминалом дальше: когда в инструкции написано «выполните команду» — нажмите кнопку Copy у нужной команды на этой странице, переключитесь в Терминал, вставьте (Cmd+V), нажмите Enter, дождитесь окончания (когда курсор снова замигает на пустой строке).

Подсказка В правом верхнем углу каждого блока с командой на этой странице есть кнопка Copy. Это самый надёжный способ перенести команду в Терминал — в буфер попадёт ровно текст команды, без лишних символов разметки или невидимых пробелов.

1.2. Установите Homebrew

Homebrew — это магазин программ для Mac, через который удобно ставить нужные инструменты. Выполните команду:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Терминал может спросить пароль — это пароль от Mac (тот же, что для входа в систему). При вводе на экране ничего не будет видно (даже звёздочек) — это нормально, просто наберите вслепую и нажмите Enter. Дождитесь окончания установки.

1.3. Подключите Homebrew к Терминалу

Чтобы Терминал заметил только что установленный Homebrew, выполните по очереди две команды:

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"

1.4. Установите ffmpeg

ffmpeg — это программа, которая умеет работать с видео. Скрипт использует её, чтобы склеивать файлы.

brew install ffmpeg

Дождитесь окончания установки.

Готово. Этот этап больше делать не нужно — программы останутся на компьютере до тех пор, пока вы их не удалите или не переустановите систему с нуля.

Этап 2. Перенос видео на Mac

Есть два способа получить минутные файлы на Mac — выбирайте удобный; скрипт понимает оба формата имён файлов, и их можно перемешивать.

Вариант А. Через приложение Mi Home

  1. Откройте приложение Mi Home на телефоне.
  2. Выберите камеру → Playback (или Storage Management) → SD Card → All Videos.
  3. Отметьте минутные файлы, которые хотите склеить, и нажмите кнопку загрузки. Файлы попадут в галерею телефона.
  4. Перенесите файлы из галереи телефона на Mac любым удобным способом и сохраните их в одну папку (например, новую папку на Рабочем столе). Способов много: AirDrop, облако (iCloud / Google Drive / Dropbox), мессенджер с самим собой («Saved Messages» в Telegram, WhatsApp), подключение телефона кабелем. Главное — чтобы в итоге все минутные .mp4 оказались в одной папке на Mac.

Если у вас iPhone с iCloud Photos, есть короткий путь:

  1. На Mac откройте приложение Photos (Фото). Загруженные из Mi Home видео появятся там автоматически (если включена синхронизация iCloud Photos).
  2. Найдите видео, выделите их все.
  3. В верхнем меню: File → Export → Export Unmodified Original for X videos.
  4. Сохраните в одну папку на Mac.

Файлы будут называться так: 1766763309000.mp4 (это Unix-таймстемп в миллисекундах).

Вариант Б. Напрямую с SD-карты камеры

Быстрее, если файлов много или нужен большой временной диапазон.

  1. Достаньте SD-карту из камеры и подключите её к Mac (через картридер или переходник).
  2. В Finder откройте карту — внутри есть папка MIJIA_RECORD_VIDEO (постоянная запись с камеры). Рядом может быть MIJIA_RECORD_MOTION — это записи по детектору движения.
  3. Внутри MIJIA_RECORD_VIDEO лежат папки с именами вида YYYYMMDDHH — каждая содержит минутные файлы за один час записи. Последние две цифры в имени папки — это час дня (от 00 до 23):
    Имя папкиЧто в ней
    202605301430 мая 2026, с 14:00 до 14:59
    202605300030 мая 2026, с 00:00 до 00:59 (полночь)
    202605292329 мая 2026, с 23:00 до 23:59
  4. Зайдите в нужную часовую папку, выделите минутные файлы (или сразу все — отфильтруете потом). Файлы внутри названы так: 29M53S_1780082993.mp4 — где 29M53S это 29-я минута 53-я секунда часа, а 1780082993 — Unix-таймстемп в секундах. Рядом с каждым .mp4 лежит .jpeg — это превью-картинка, её скрипт игнорирует.
  5. Скопируйте выбранные файлы в одну плоскую папку на Mac (например, новую папку на Рабочем столе). Если нужно несколько часов — заходите по очереди в каждую часовую папку и копируйте оттуда нужные минуты в ту же целевую папку.

Можно копировать «с запасом» — например, целый час, если нужно полчаса. На Этапе 3 скрипт либо склеит всё, что вы скопировали, либо отфильтрует диапазон по вашему указанию.

Важно Не переименовывайте файлы — скрипт ориентируется по именам и таймстемпам в них. Если переименуете — он не поймёт, в каком порядке клеить.

Этап 3. Склейка

⚠ Самое важное — не пропустите

Два шага, на которых чаще всего спотыкаются. Их нужно делать каждый раз, когда вы заводите новую папку для очередной порции видео:

  1. Положить файл vidmerge.sh в ту же папку, где лежат минутные клипы. Файл нужно скопировать туда заново — он не следует за вами автоматически. Без него Терминал в этой папке скажет command not found.
  2. Разрешить запуск командой chmod +x vidmerge.sh. Это делается один раз для каждого нового файла vidmerge.sh, который вы скопировали в новую папку. Без этого шага Терминал скажет Permission denied.

Это самые частые ошибки, поэтому если что-то не работает с первой попытки — проверьте именно эти два пункта.

Весь процесс в одной картинке:

шаг 1. моя_папка/ ← ваша папка с видео ├─ 14_30_00.mp4 ├─ 14_31_00.mp4 ├─ 14_32_00.mp4 └─ ... (минутные файлы) │ ▼ скопируйте сюда файл vidmerge.sh │ шаг 2. моя_папка/ ├─ 14_30_00.mp4 ├─ ... └─ vidmerge.sh ← скопировали │ ▼ откройте Терминал в этой папке │ (правый клик по папке → «Новый терминал в папке») │ ════ шаги 3 и 4 — наберите (или вставьте) в открывшемся Терминале ════ │ шаг 3. chmod +x vidmerge.sh ← разрешили запуск (один раз) │ ▼ │ шаг 4. ./vidmerge.sh ← запустили склейку │ ▼ ▓▓ ✓ Done — that worked

3.1. Положите скрипт рядом с видео

Скопируйте файл vidmerge.sh в ту же папку, где лежат ваши минутные видео — перетащите его в Finder, либо скопируйте через Cmd+C / Cmd+V. Это нужно делать каждый раз для каждой новой папки: если вы перенесёте видео в другое место, скрипт туда тоже надо скопировать.

3.2. Откройте Терминал в этой папке

Простой способ (рекомендуем):

  1. В Finder кликните по папке с видео правой кнопкой мыши.
  2. Выберите «Новый терминал в папке» (или New Terminal at Folder).
  3. Откроется Терминал, уже находящийся в нужной папке.

Если такого пункта в меню нет — его нужно включить один раз. Путь зависит от того, на каком языке у вас интерфейс macOS:

  • Русский интерфейс: Системные настройки → Клавиатура → Сочетания клавиш → Службы → Файлы и папки → поставить галочку у пункта «Новый Терминал в папке».
  • English interface: System Settings → Keyboard → Keyboard Shortcuts → Services → Files and Folders → check “New Terminal at Folder”.

На старых версиях macOS (до Ventura) этот раздел называется System Preferences / Системные настройки и устроен немного иначе, но искомый пункт там есть — ищите по слову «Terminal» в Services / Службы.

Запасной способ через команду cd:

  1. Откройте Терминал любым способом.
  2. Наберите cd (с пробелом в конце).
  3. Перетащите папку с видео из Finder в окно Терминала — путь подставится автоматически.
  4. Нажмите Enter.

Проверить, что вы попали в нужную папку, можно командой:

ls *.mp4

Должны увидеть список своих минутных файлов и скрипт.

3.3. Разрешите запускать скрипт

В Терминале выполните:

chmod +x vidmerge.sh

Эта команда говорит системе: «этот файл — программа, его можно запускать». Достаточно сделать один раз для файла vidmerge.sh в этой папке.

Запомните Если в следующий раз вы скопируете файл vidmerge.sh в другую папку (с другими видео) — там тоже нужно один раз выполнить chmod +x vidmerge.sh. Mac хранит разрешение запуска отдельно для каждого файла, оно не переносится при копировании. Без этого шага при попытке запустить вы увидите Permission denied.

3.4. Запустите скрипт

./vidmerge.sh

./ в начале означает «программа из этой же папки» — без этого Терминал не поймёт, что нужно искать программу здесь.

По умолчанию скрипт склеивает все распознанные минутные файлы из папки. Это и есть основной сценарий, если вы уже отобрали нужные файлы руками при копировании.

Сначала появится небольшая обложка с ASCII-иконкой камеры — это просто оформление, а затем — превью того, что собирается сделать:

В ответ:

  • Просто Enter — склеить с показанными настройками.
  • Введите любой текст — он станет новым комментарием. Превью обновится с новым именем выходного файла, и вопрос задастся снова. Удобно, если дефолтный комментарий (имя папки) не подходит.
  • q + Enter — отмена, никаких файлов не создаётся.

3.5. Опционально: фильтр по времени

Если в папке оказались лишние файлы, и нужно склеить только часть из них по времени — добавьте флаги --from и --to (или --from + --duration):

./vidmerge.sh --from "сегодня 14:00" --to "сегодня 15:30"
./vidmerge.sh --from "14:00" --duration 1h30m
./vidmerge.sh --from "вчера 22:00" --to "сегодня 02:00"

Форматы времени:

Что ввестиЧто это значит
сегодня 14:00Сегодня в 14:00
вчера 22:00Вчера в 22:00
14:00Просто время = сегодня в 14:00
2026-05-30 14:00Конкретная дата и время

По-английски тоже работает: today 14:00, yesterday 22:00.

Длительность: 90m (90 минут), 2h (2 часа), 1h30m (час тридцать), 45s (секунд).

В формате даты между числами — обычные дефисы (-), не длинные тире и не подчёркивания.

3.6. Дождитесь окончания

Сама склейка очень быстрая (несколько секунд на час видео — потому что копируется поток без перекодировки). Скрипт покажет короткую строку прогресса и затем итог:

▸ Merging files… ░ ░ ░ → ▓▓▓ ▓▓ ✓ Done — that worked File: merged_2026-05-30_14-00_to_15-00_Lidl.mp4 Duration: 1h 00m 00s Size: 1.2 GB Reveal in Finder? [Enter=yes, n=no]:

В последней строке вопрос про Finder:

  • Enter — откроет Finder с подсвеченным готовым файлом, и скрипт попрощается: Off you go.
  • n + Enter — Finder не открывает, скрипт скажет: Alright then.

Готовое видео лежит в той же папке, где минутные файлы. Если случайно отказались открыть Finder — двойной клик на файле в этой папке открывает его в стандартном плеере. Или вручную из Терминала (подставьте реальное имя файла):

open merged_2026-05-30_14-00_to_15-00_Lidl.mp4

Как сделать следующую склейку

Программы (Homebrew, ffmpeg) уже установлены — этот этап повторять не нужно. У вас на компьютере где-то лежит файл vidmerge.sh — это тот же файл, который вы использовали в первый раз. Каждый раз, когда нужно склеить новую порцию видео, повторяйте вот эти шаги:

  1. Соберите новые минутные файлы в одну папку. Через приложение Mi Home или напрямую с SD-карты — оба способа описаны в Этапе 2.
  2. Положите в эту же папку файл vidmerge.sh. Это обычный файл — перетащите его мышкой в Finder, как любой документ. Удобно хранить эталонную копию vidmerge.sh где-то у себя (например, на Рабочем столе или в Документах) и каждый раз копировать её в новую папку с видео.
  3. Откройте Терминал в этой папке: правый клик по папке в Finder → Новый терминал в папке.
  4. Если вы только что скопировали vidmerge.sh в эту папку — один раз выполните в Терминале chmod +x vidmerge.sh. Эта команда разрешает запускать файл как программу. Если эта папка осталась с прошлого раза и файл vidmerge.sh уже лежал в ней — этот шаг уже сделан раньше, повторять не нужно.
  5. Запустите склейку командой ./vidmerge.sh. Нажмите Enter, когда скрипт спросит подтверждение.
  6. Если нужно склеить только часть файлов по времени — добавьте флаги --from и --to (см. раздел «Опционально: фильтр по времени»).
Повторим — это главное

Если что-то не запускается с первой попытки — почти всегда причина одна из двух:

  • Файла vidmerge.sh нет в той же папке, что и видео. Терминал ответит: command not found. Решение: перетащите файл в нужную папку.
  • Файл есть, но запускать его не разрешено. Терминал ответит: Permission denied. Решение: один раз выполните chmod +x vidmerge.sh.

Сначала проверьте эти два пункта — почти наверняка дело в одном из них.

Если время на камере отличается от вашего

Например, у камеры в настройках стоит часовой пояс Москвы, а вы в Берлине — тогда «сегодня 14:00» на Mac будет означать одно, а в имени файла — другое. Проверить это можно одной командой:

./vidmerge.sh --verify-tz

Скрипт возьмёт один файл и сравнит время в имени файла со временем, когда файл реально оказался на диске. Если совпадает — скажет Timezone OK (можно спокойно пользоваться сегодня/вчера). Если есть расхождение — покажет, на сколько секунд оно сдвинуто, и предупредит, что для команд с --from/--to нужно учитывать это смещение.

Продвинутый режим

У скрипта есть несколько полезных режимов помимо дефолтного. Каждый — это сценарий «что я хочу сделать» и команда, которую для этого нужно выполнить.

Склеить всё, что лежит в папке (поведение по умолчанию)

./vidmerge.sh

Склеить только заданный диапазон времени

./vidmerge.sh --from "сегодня 14:00" --to "сегодня 15:30"

Склеить от указанного момента на заданную длительность

./vidmerge.sh --from "сегодня 14:00" --duration 1h30m

Склеить ночную запись через полночь

./vidmerge.sh --from "вчера 22:00" --to "сегодня 02:00"

Задать своё имя выходного файла

./vidmerge.sh --from "14:00" --duration 1h --output entrance_morning.mp4

Задать свой комментарий (перебить имя папки)

./vidmerge.sh --comment "Курьер у двери"

Склеить совсем без комментария (только дата в имени файла)

./vidmerge.sh --comment ""

Запустить скрипт для папки, которая находится в другом месте

./vidmerge.sh --dir /Users/имя/Desktop/cam_kitchen

Не задавать вопросов на подтверждение (для автоматизации)

./vidmerge.sh -y

Показать полный технический вывод ffmpeg (битрейт, прогресс, варнинги)

./vidmerge.sh --verbose

Посмотреть весь список параметров скрипта

./vidmerge.sh --help

Если что-то пошло не так

command not found: brew — Homebrew не установлен или не подключён к Терминалу. Вернитесь к шагам 1.2 и 1.3.

command not found: ffmpeg — нет ffmpeg. Выполните brew install ffmpeg.

Permission denied при запуске ./vidmerge.sh — Mac пока не разрешает запускать этот файл как программу. Один раз выполните в Терминале chmod +x vidmerge.sh — это даст разрешение, и команда сработает.

⚠No .mp4 files in … — Терминал смотрит не в ту папку. Проверьте ls *.mp4 — должны быть видны ваши минутные файлы. Если не видны — вы не в той папке.

⚠Skipped files with unrecognized names — в папку попал .mp4, имя которого не похоже ни на формат Mi Home Export (1766763309000.mp4), ни на формат SD-карты (29M53S_1780082993.mp4). Скрипт его проигнорирует и продолжит работу. Если это файл, который вы переименовали — верните оригинальное имя.

⚠No files matched the selected range — под фильтр по времени не попало ничего. Возможно, время в файлах сдвинуто относительно вашего — запустите ./vidmerge.sh --verify-tz. Или диапазон задан вне реальных файлов.

⚠Video parameters differ between files — клипы записаны с разными настройками камеры (разное разрешение или fps). Быстрая склейка тут не сработает. Чаще всего это значит, что в папку случайно попали файлы от другой камеры или из другой серии — почистите папку и попробуйте снова.

⚠Comment too long — введённый комментарий длиннее 50 символов. Скрипт скажет «Bit much.» и попросит ввести покороче — просто наберите более короткую версию и нажмите Enter.

Видео не открывается / битое — обычно та же причина, что выше («Video parameters differ»). Скрипт перед склейкой должен предупредить, но если файлы прошли проверку и результат всё равно битый — напишите автору или попробуйте перезапустить.

Несколько камер одновременно

Не смешивайте файлы от разных камер в одной папке — у них могут быть разные параметры (разрешение, fps), и склейка либо не сработает, либо даст артефакты.

Сделайте отдельную папку под каждую камеру и запустите скрипт для каждой отдельно. Файл vidmerge.sh можно скопировать в каждую папку (не забыв в каждой выполнить chmod +x vidmerge.sh), либо хранить файл в одном месте и запускать его оттуда с параметром --dir, указывая путь к нужной папке с видео.
